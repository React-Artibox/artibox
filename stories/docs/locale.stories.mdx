## Installation

```bash
npm install @artibox/locale --save

or

yarn add @artibox/locale
```

## Introduction

This package just only includes `all locales for artibox` and the interface [LocaleDefinition](https://github.com/React-Artibox/artibox/tree/master/packages/locale/src/typings.ts).  
All locales are restricted in the definition and each locale is just an object.

## Usage

Please import each locale since there is no root file.

```ts
import { enUS } from '@artibox/locale/en-US';
import { zhTW } from '@artibox/locale/zh-TW';
```
