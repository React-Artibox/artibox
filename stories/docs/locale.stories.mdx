# @artibox/locale

[![npm package](https://img.shields.io/npm/v/@artibox/locale.svg?maxAge=60)](https://www.npmjs.com/package/@artibox/locale)
[![npm downloads](https://img.shields.io/npm/dt/@artibox/locale.svg?maxAge=60)](https://www.npmjs.com/package/@artibox/locale)

## Installation

```bash
npm install @artibox/locale --save

or

yarn add @artibox/locale
```

## Introduction

This package just only includes `all locales for artibox` and the interface `LocaleDefinition`.

Please see the [definition](https://github.com/React-Artibox/artibox/tree/master/packages/locale/src/typings.ts) first.  
All locales are restricted in this definition and each locale is just an object.

## Usage

Please import each locale since there is no root file.

```ts
import { enUS } from '@artibox/locale/en-US';
import { zhTW } from '@artibox/locale/zh-TW';
```

### For React

Provide locale by using `ConfigsProvider`.

```tsx
import { ConfigsProvider } from '@artibox/slate-react/configs';
import { enUS } from '@artibox/locale/en-US';

...

<ConfigsProvider locale={enUS}>
  ...
</ConfigsProvider>
```

For editor, just use the `Artibox` component built w/ `ConfigsProvider`.

```tsx
import { Artibox } from '@artibox/slate-react';
import { enUS } from '@artibox/locale/en-US';

...

<Artibox locale={enUS}>
  ...
</Artibox>
```
